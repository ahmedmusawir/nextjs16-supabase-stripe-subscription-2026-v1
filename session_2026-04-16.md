## Lessons Learned
- Every route needs a `loading.tsx` — no exceptions. Parent loading.tsx does NOT cascade to nested routes in Next.js App Router.
- ShadCN Button `default` variant has no border. Always add explicit border classes.
- Saved feedback memory for both issues so they don't recur.
