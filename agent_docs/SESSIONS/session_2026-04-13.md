## Key Decisions / Patterns (for any agent picking this up)

**Next.js 15 dynamic params — ALWAYS await:**
```ts
interface Props { params: Promise<{ id: string }> }
const Page = async ({ params }: Props) => {
  const { id } = await params;
```

**Admin client (service_role — bypasses RLS):**
```ts
import { createAdminClient } from "@/utils/supabase/admin";
```

**Regular client (anon key — respects RLS, for own-user operations):**
```ts
import { createClient } from "@/utils/supabase/server";
const supabase = await createClient();
```

**Role colors (consistent across all portals):**
```ts
superadmin → text-purple-600 dark:text-purple-400
admin      → text-red-600   dark:text-red-400
member     → text-green-600 dark:text-green-400
```

**Input/Button/Label styling:**
```
Input:  p-6 bg-slate-100 dark:bg-slate-500 dark:text-white
Button: w-full bg-slate-700 text-white hover:bg-slate-900 ...
Label:  uppercase text-xs font-bold text-zinc-500 dark:text-white
```
